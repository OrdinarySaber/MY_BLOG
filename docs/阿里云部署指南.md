# 阿里云服务器部署指南

## 部署架构

```
用户浏览器
     │
     ▼
┌────────────────────────────────────┐
│         阿里云服务器                │
│  ┌──────────────────────────────┐ │
│  │    ASP.NET Core Web 应用      │ │
│  │  端口: 80 (HTTP)              │ │
│  │  ├── 静态文件服务 (前端)        │ │
│  │  └── REST API (后端)          │ │
│  └──────────────────────────────┘ │
│  数据: JSON 文件存储                │
└────────────────────────────────────┘
```

## 第一步：服务器环境准备

### 1.1 安装 .NET Runtime

在阿里云服务器上执行以下命令安装 .NET 10 SDK：

```bash
# 添加 Microsoft 包仓库
wget https://packages.microsoft.com/config/ubuntu/24.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb

# 安装 SDK
sudo apt-get update && sudo apt-get install -y dotnet-sdk-10.0
```

或者安装 Runtime 即可（不开发）：

```bash
sudo apt-get update && sudo apt-get install -y dotnet-runtime-10.0
```

### 1.2 安装 Nginx (可选，反向代理)

```bash
sudo apt-get install -y nginx
```

## 第二步：上传项目文件

### 方式一：使用 Git（推荐）

1. 将项目推送到 GitHub/Gitee
2. 在服务器上克隆：

```bash
cd /var/www
git clone https://github.com/你的用户名/BlogProject.git
```

### 方式二：使用 FTP/SFTP

使用 FileZilla 或其他 FTP 工具上传整个项目文件夹到服务器 `/var/www/BlogApi`

## 第三步：配置防火墙

```bash
# 开放端口
sudo ufw allow 80    # HTTP
sudo ufw allow 443   # HTTPS
sudo ufw allow 5096  # 开发测试用
sudo ufw enable
```

## 第四步：运行项目

### 4.1 开发模式测试

```bash
cd /var/www/BlogApi
dotnet run
```

访问 `http://你的服务器IP:5096`

### 4.2 生产环境运行

```bash
cd /var/www/BlogApi
ASPNETCORE_ENVIRONMENT=Production dotnet run --urls "http://0.0.0.0:80"
```

### 4.3 后台运行服务

使用 Systemd 管理服务：

```bash
sudo nano /etc/systemd/system/blogapi.service
```

写入以下内容：

```ini
[Unit]
Description=Blog API Application
After=network.target

[Service]
WorkingDirectory=/var/www/BlogApi
ExecStart=/usr/bin/dotnet /var/www/BlogApi/bin/Release/net10.0/BlogApi.dll --urls "http://0.0.0.0:80"
Restart=always
RestartSec=10
User=www-data
Environment=ASPNETCORE_ENVIRONMENT=Production

[Install]
WantedBy=multi-user.target
```

启动服务：

```bash
sudo systemctl daemon-reload
sudo systemctl start blogapi
sudo systemctl enable blogapi
```

## 第五步：配置 Nginx 反向代理（可选）

编辑 Nginx 配置：

```bash
sudo nano /etc/nginx/sites-available/default
```

修改为：

```nginx
server {
    listen 80;
    server_name 你的域名或IP;

    location / {
        proxy_pass http://localhost:80;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection keep-alive;
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

重载 Nginx：

```bash
sudo nginx -t
sudo systemctl reload nginx
```

## 第六步：配置域名（可选）

1. 在阿里云控制台添加域名解析
2. 指向服务器公网 IP

## 快速部署命令汇总

```bash
# 1. 安装 .NET
wget https://packages.microsoft.com/config/ubuntu/24.04/packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
sudo apt-get update && sudo apt-get install -y dotnet-runtime-10.0

# 2. 克隆项目
cd /var/www
git clone <你的项目地址>

# 3. 防火墙
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp

# 4. 启动服务
cd /var/www/BlogApi
ASPNETCORE_ENVIRONMENT=Production dotnet run --urls "http://0.0.0.0:80"
```

## 常用管理命令

```bash
# 查看日志
sudo journalctl -u blogapi -f

# 重启服务
sudo systemctl restart blogapi

# 停止服务
sudo systemctl stop blogapi

# 查看状态
sudo systemctl status blogapi
```

## 注意事项

1. 首次运行会自动创建管理员账户：`admin / admin123`
2. 数据文件位置：`/var/www/BlogApi/BlogApi/`
   - users.json（用户数据）
   - sessions.json（会话数据）
   - comments.json（评论数据）
3. 建议定期备份这些 JSON 数据文件
